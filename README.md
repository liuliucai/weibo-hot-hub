    - 商标注册应该遵循哪些原则？我国商标法第七条第一款规定，申请注册和使用商标，应当遵循诚实信用原则。我国商标法实行商标注册制，由申请人申请注册从而取得商标专用权。申请人申请注册和使用商标应当遵循诚实信用原则。此外，商标申请审查遵循的是申请在先、自愿注册等原则。
